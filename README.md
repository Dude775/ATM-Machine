## תיאור הפרויקט

בנינו מכונת ATM מלאה עם ממשק גרפי בטינקר.
אפשר להתחבר כמשתמש רגיל או כמנהל מערכת.
כל הנתונים נשמרים בקובץ JSON כך שהמידע נשמר בין הפעלות.

---

## איך מריצים

```bash
python main.py
```

> **חשוב:** אין צורך בהתקנת ספריות חיצוניות. הכל מובנה ב-Python.

## פיצ'רים

### משתמש רגיל
- התחברות עם מספר חשבון + קוד PIN
- הפקדת כסף
- משיכת כסף (עם בדיקת יתרה)
- העברה לחשבון אחר — נרשמת בשני הצדדים
- צפייה בהיסטוריית עסקאות
- שינוי קוד PIN

### מנהל מערכת
- התחברות עם סיסמת מנהל
- יצירת חשבון חדש
- צפייה בכל החשבונות
- חסימה / שחרור חשבון

## דברים שחקרנו לבד

- עבודה עם `json` ב-Python (קריאה וכתיבה)

- ניהול מסכים ב-Tkinter עם `tk.Frame`
- נעילה אוטומטית אחרי 3 ניסיונות PIN שגויים
