# ATM Machine

This is our ATM machine project for the Python course. It has user login, deposit, withdraw, transfer, and admin panel.

**Partners:** David Rubin & Daniel Yakov

## How to run

```
python main.py
```

## Files

- `main.py` - starts the program
- `models.py` - Account and Bank classes
- `storage.py` - save and load data from JSON
- `ui.py` - all the screens and buttons (tkinter)
- `README.md` - this file

> data.json is created automatically on first run
